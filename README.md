# Lab-IA-Generativa
Repositorio del proyecto del Laboratorio de IA Generativa. El proyecto se basa en el diseño e implementación de un Sistema RAG Multimodal para Asistencia Universitaria Basado en Documentos Textuales y Contenido Visual.

## Requisitos

1. Instalar dependencias Python:
pip install -r requirements.txt

2. Instalar Ollama:
https://ollama.com/download

3. Descargar modelos:
ollama pull llama3.1:8b
ollama pull llava

4. Ejecutar Ollama:
ollama serve
